Python Functions

Find yourself doing something repeatedly when coding in Python? Put it in a function!

Defining the function

Begin by using the def keyword, followed by the function name, and then parentheses and a colon. Put any function variables you want to pass to the function in the parentheses.Now enter any code you want to have in your function, indented one tab over from where your function is defined.

Calling the function

You can now call the function by typing the function name followed by parentheses and any variables you want to pass to the function. For example, if your function is named 'foo', you can call it in your code like this: foo()

There is a lot more to learn about Python functions, try starting here!
